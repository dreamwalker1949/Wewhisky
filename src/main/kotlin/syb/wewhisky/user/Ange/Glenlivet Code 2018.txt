正文：虽然我从来就没说过Glenlivet一句好话，但侥幸得了个小酒版礼盒，还是谢谢商家......给了他们自己一次重新证明自己的机会（？？？），毕竟一年一度的code也算个提升酒厂形象、捎带竞争一下中端市场的小网红了，前几年的cipher、alpha都没接触过，要是被我的主观印象给黑了那也只能怪你们的年份款做的太灭人欲：
abv: 48%。
参考价: ￥1300/750ml。
Eye: 麦芽金。
Nose: 姜饼、橘子汽水、牛奶巧克力、风油精，杯醒5分钟后出现分量恰到好处的炭烤麦芽香，梳打和苹果派，香薰蜡烛、烤苹果木的灰烬。
Palate: 核桃仁、开心果、苹果脯、青梅、青芒果、姜汁啤酒，醒酒后出现香蕉布丁、梳打饼干、可丽饼和卤味拼盘。神似Clynelish的蜡油味，短促但清晰的辛辣和灼烧感随着杯醒被大幅度缓解。
Balance, Body: 酒体中等偏薄， 蜡质感明显，干燥青涩，15年上下的熟成年龄比较合理。酒力出的明显但后劲较疲乏，也没有看出任何修饰酒精感的意图和作为。
Finish: 中长，青梅、橡木桶、风干的橙皮、辛香料、姜汤。
Score Breakdown
Nose: 85
Palate:86
Balance, Body: 82
Finish: 84
Summary: 84/100
Overall: 相当勉强的卡到了及格线，又一款相当“斯宾塞”的中低年份酒款。尽管已经是搞了一些花样（我猜测的桶型马德拉、波特、霞多丽桶之中的其一或其二），尽管已经算是喝过的Glenlivet里第二能接受的酒款，但即便如此本周之内我就会忘掉它。Glenlivet之于我的印象永远就是一个你“相敬如宾”又恰好卡在“七年之痒”上下的老伴儿：你说激情吧，那肯定没了；你说嫌弃吧，还真没到那份上；稍客气显生分，稍腻歪显不正经。但是有一点不接受反对意见：同样是一身黑，密码的瓶子比星图是好看的。
