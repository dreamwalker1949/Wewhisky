abv: 57.8%。
参考价: ￥2500/700ml。
Eye: 浅铜色。
Nose: 扑鼻的黄油、香蕉、蜂蜜、炙烤焦糖、铜器、炖水果、热红酒，杯醒十分钟后显著的芒果、肉桂卷。
Palate: 爆炸开的口感，同一股肥美圆润的油脂裹挟里，树莓酱、蜂蜜厚多士伴着芒果糯米糍，海盐焦糖玛奇朵融着淡奶油、水蜜桃奶昔，香草、橡木桶、肉桂、陈皮等辛香料托底，建筑起复杂充实庞大的风味体。
Balance, Body: 酒力强、酒体重，但是一丝多余的赘肉也没见，反而是充满香腻丰腴的性感。风味的搭配主次分明详略得当，平衡感极佳。
Finish: 长。黄油、香蕉、热红酒、熟透的水蜜桃、肉桂、橡木桶、姜饼。
Score Breakdown
Nose: 94
Palate:95
Balance, Body: 91
Finish: 92
Summary: 93/100
Overall: 闻香强劲，引人入胜；口感爆炸绚丽，目接不暇；回味悠长，余香经久不散。丰腴香艳又不失坚毅清奇，可以称得上是遒劲扎实、又浪又稳。在任何筛选条件和比较环境中，Dair Ghaelach Tree 8都是个人向中能进入第一档（前五）的酒款。
