标题：金车噶玛兰 Kavalan Distiller’s Select 珍选
链接：http://wewhisky.com/forum.php?mod=viewthread&tid=12321
发布时间：2019-01-04 14:24
正文：第一次尝试来自宝岛台湾 金车噶玛兰 酒厂的威士忌。本来想着从性价比最高的独奏系列入手，奈何 价 还是有点高，只好放低一些对 性 的要求，还是先来一瓶最入门款好了。
闻香：84
浓郁香甜。倒入杯中，成熟水果香气浓郁，哈密瓜，菠萝，水蜜桃，黑布林，水果硬糖，香草冰淇淋，果脯，淡淡的茉莉花香。静置数分钟，香气愈发甜腻，枫糖浆，蜂蜜，甜奶油，柠檬蛋糕。整体柔和浓郁甜美，毫无酒精刺激，但缺乏层次感且略显单调。
酒体：83
深金黄色，被过度氧化的花生油的金黄。中度酒体，晃动酒杯挂杯度低，油脂感低，整体偏水润。
品味：83
轻盈淡雅。入口冰凉水润，毫无酒精刺激，甜美，淡淡的花香，成熟水果的香甜，蜂蜜的甜腻，全麦面包的麦香，青草的芳香。在口腔打转，果脯，话梅，柠檬皮，甘草合剂，酸酸甜甜的感觉。入喉稍有刺激，口腔发干，余味短，只有淡淡的甜味。
综合评分：83
还是得说一分钱一分货，但是200块左右的价格能买到这样平衡易饮，除了稍微淡一点以外没有明显短板的酒已经实属难得。对于我来说，纯饮可能还是会选择味道更为浓郁的酒款，但这款珍选有一个非常适合它的场合：
配餐。无论是吃牛杂还是在街边吃快餐，这瓶珍选都能和各种菜系搭配，解腻解乏，衬托菜色，还绝不抢主菜风头。这就是理想的餐酒应该做到的事情。如果把它定义为餐酒，我可以给它87分。
