正文：又是爱尔兰，但因为是单一麦芽，倒算是久违了。感谢@Oneyunqi 支持的分享瓶，wshnb!!!!!一般来说爱尔兰的单一麦芽通通认为是Cooley。不光因为老饕们觉得twa的单一麦芽像是Cooley，TWA的产品线也给了佐证——其旗下的爱尔兰单一麦芽最早出现的vintage是1988——Cooley正式生产单一麦芽的元年。没有一桶，哪怕一桶TWA爱尔兰，是早于这个年份的。更不用提Bushmills和ib少有合作，而Midleton的单一麦芽简直难得一见了。我并不会因为是Cooley就否定一款爱尔兰，Cooley出色的作品是十分让我折服的。但品鉴Cooley的时候，我开诚布公地说，会不断提醒自己在喝一款斯佩塞威士忌。那么这对兄弟，一款是15-16年，一款14-15年，表现如何？
Irish Single Malt TWA 2002-2018
类别: 爱尔兰单一麦芽威士忌。
abv: 49.8%。
参考价: ￥1300-1500/700ml。
Eye: 淡金色。
Nose: 无泥煤的波本桶，柠檬、柚子酱、蜂蜜面包、铜器式样的矿物感溢出，有些像清丽点的pinot grigio了。
Palate: 香蕉船、苹果西达、秋梨膏、蜂蜜柚子茶、橡木气息，又是典型的斯佩塞式波本桶风味。这么说也没毛病，无论从生产流程、味道哪个角度看，Cooley都和大家印象中的爱尔兰毫无关系，说斯佩塞是没毛病的。
Body: 中度偏向轻盈，有灼烧，维度算不上多，果香清冽、木香适度。
Finish: 橡木苦、西柚、烘焙点心。强而短。
Score Breakdown
Nose: 83
Palate: 84
Finish: 83
Balance, Body: 80
Summary: 83/100
Irish Single Malt TWA 2001-2016
类别: 爱尔兰单一麦芽威士忌。
abv: 51.9%。
参考价: ￥1300-1500/700ml。
Eye: 琥珀色。
Nose: n填雪莉。倒在杯子里初闻是焦糖玛奇朵、杏干、咖啡蛋糕香气，浓郁讨喜。让我不禁怀疑这是Bushmills而不是Cooley。不过醒后还是挺失望的——它回归了雪莉桶的硫磺、火柴盒子香气，以及一丝焦肉、瓜子臭子。并不是说有一定硫磺气的雪莉桶是差酒，但硫磺本身气味令人不适，在这项上过于包容是对无硫磺气雪莉桶的不公，所以上限是要有的。
Palate: 部分还算有些维度。硫磺比闻香减弱但仍存在，糖渍橄榄、鲜杏、小红莓干味道较为明显，后段转为火柴、焦油、烧烤猪肉——尤其像是菲律宾时过火的chicharron。
Body: 中度酒体，刺激感于上款波本桶相仿。复杂度可观，平衡性合格。
Finish: 中强且中长。糊肉、焦油、而杏干紧随其后。
Score Breakdown
Nose: 80
Palate: 86
Finish: 84
Balance, Body: 84
Summary: 84/100
Overall: 2002这款酒算是不错的波本桶，如果是桶强的话我比较惊讶，到底什么水平的库存管理能把十几年的酒精度做成这个样子，指定不是岛屿熟成。酒不算太复杂，完成度一般般，分数不是很高。不过果味浓郁、清新，实际上是很适合女孩子的小甜水。
2001应该是n填雪莉桶，不算太轻松。事实上很像林一峰选桶的那款PX桶爱尔兰，只是硫磺味少一些，复杂度高一点。虽然不一定易饮，但比起所谓OB的14年Cooley——Teeling Revival III还是完胜的。有几个瞬间让人十分自我怀疑，太像Bushmills了，尤其是我手边恰好有一瓶16的时候。
如果作为IB的调剂，2001应该是值得一尝的。Cooley这个厂一直处在一个矛盾当中：它为成为非典型爱尔兰而出生，又因为非典型而被排斥，但威士忌这东西好玩儿也就在于此。
