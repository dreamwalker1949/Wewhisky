标题：山崎2011波本桶(Yamazaki Bourbon Barrel 2011)酒评#246
链接：http://wewhisky.com/forum.php?mod=viewthread&tid=11775
发布时间：2018-12-12 23:56
正文：又是山崎，嘿嘿嘿。2011年的波本桶nas是比较早期的一批山崎风味桶了，当时应该是一百美元左右。尽管从前被大家误认为是波本桶的原教旨支持者，我还是要说一句，波本桶是最挑酒的桶型。虽然其实我喜欢的是各种不是雪莉的强化酒桶以及混桶风格，我也一直坚信波本桶对于酒质的表现是最直接、最真实的。既然如此，多桶兑和的山崎波本桶，想给我们看的那一面山崎就格外引人入胜。
类别: 日本单一麦芽威士忌。
abv: 48%。
参考价: ￥5500-6000/700ml。
Eye: 金黄色。
Nose: 浓重香草、纸杯蛋糕、焦木、甜橙、酸面包。很像纯粹的波本威士忌。
Palate: 香草、砂糖、奶味乐天汽水和柠檬干，与闻香相仿。多了点生榛子。复杂度非常低、非常低、非常低。不过作为波本爱好者我觉得还挺好喝的。
Body: 中度酒体，酒液有黏性，厚度可观。但辣、贼拉甜，平衡感不敢恭维。
Finish: 短，香草荚、榛子。
Score BreakdownNose: 81
Palate: 81
Finish: 78
Balance, Body: 81
Summary: 80/100
Overall: 回家对比了一下自己对Michter's的评分，把山崎波本桶从78提到了80，我还挺喜欢一小口一小口咂么这样的波本桶的。复杂度？表现力？惊喜？都不要想了，一点没有。某些程度上更像是高质量的涮桶水，调配得还算是不错，可惜离开了日本橡木与雪莉桶后，这款接近50度的山崎好像完全没把展现酒厂特性的事情放在心上，就那样做着一个普通的波本桶，这样的低分也算是难堪了。之所以这次对三得利不太harsh是因为这锅真不是他们的，谁叫日吹什么垃圾都肯接呢？这么一想这款被大家赋予众望的山崎反而有些可爱了。
即便如此我个人仍然很喜欢这种风格，我希望山崎能多做这种风格的酒，并且把价格定在240元左右，也就是现在的1/20，谢谢三得利。
