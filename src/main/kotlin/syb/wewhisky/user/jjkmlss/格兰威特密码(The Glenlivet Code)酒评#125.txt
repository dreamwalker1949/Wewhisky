正文：感谢@Ange 赠酒，Glenlivet家前两个谜之系列我都没有喝，密码也是浅尝辄止，但是这款酒非常有意思，还是忍不住想跟大家分享一下——除了独特，独特，还是独特。Glenlivet曾是斯佩塞威士忌的巅峰，就算现在大佬追捧的麦卡伦，也曾经试图蹭Glenlivet的热点。不过斯佩塞的口感与味觉向来稳定但不极端，希望这款“匿名”威士忌，可以让这个印象有所改观。
abv: 48%。
参考价区间: ￥1000-1500/700ml。
Eye: 淡金色。
Nose: 理所当然，非常地Glenlivet。闻到了典型的蜂蜜、吐司、柑橘、苹果、蓝莓麦芬。
Palate: 蜂蜜、柑橘、巧克力、新橡木桶、生姜、甘草、梳打饼干、石灰，从尝味角度上，似乎有一些新桶和淡泥煤的特性。
Balance, Body: 中度酒体，略有灼烧，斯佩塞绝佳的平衡感。
Finish: 长。灰烬、梳打饼干混着未去皮的卡芒贝尔奶酪、牛奶巧克力。
Score Breakdown
Nose: 85
Palate:88
Balance, Body: 85
Finish: 88
Summary: 87/100
Overall: 所以最后我认为除了正常的波本桶以外，密码有橡木新桶、泥煤原酒和马德拉桶的风味在其中，而酒龄在13-18年左右。酒质不错了，The Glenlivet我从12喝到21，一款没有错过过，我不相信25会有什么重大突破，但这款密码应该是目前最喜欢的格兰威特，很出色，如果不是这种近乎盲品的方式，我想我会以一种全新的角度评价它。
