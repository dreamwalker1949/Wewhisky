标题：The Last Drop 50yo酒评一则
链接：http://wewhisky.com/forum.php?mod=viewthread&tid=8299
发布时间：2018-07-06 22:29
正文：首先感谢Orosis大佬慷慨赠与分享瓶，让我能够喝到这么牛的一支酒。
The Last Drop专营世上最矜罕的烈酒，并致力从苏格兰以及世界各地偏远、最优秀的仓库和酒窖发掘独特的上乘美酒。品牌一直严谨挑选历史悠久、且深邃浓郁的佳酿。
以上是国内公众号对这个牌子的解读，对我而言，从未奢望能够喝到这种级别的威士忌，O壕主动赠酒，我的感激之情难以言表。The Last Drop出品，必属精品，这已经形容得不够准确了，他们出的酒，几乎都是绝品。
这一支酒是The Last Drop于2015年装瓶的50年调和威士忌，酒精度数51.8度，一共装了898瓶。WB上显示其中的酒液在7个oloroso雪莉猪头桶中熟成了20年，WB评分92.75分。我的拙评如下：
C：金色
N：香水气息混杂着桃子和红糖，之后有一点不是特别浓郁的水果，一点点水果硬糖和杏仁，之后肉桂的味道明显，总之是比较综合的香料、香水和水果的结合。
T：入口柔滑又活泼，包裹感很强，迅速覆盖口腔，这一点很明显，开始的味道是焦糖的甜味，甜度适中，伴随着口腔上颚的香水味道袭来，之后是一点点水果和令人愉悦的巧克力蛋糕的口感，一点点姜汁、胡椒、肉桂，之后是一点非常轻微的香料味道结尾。
F：尾韵较长，主要是红糖的甜味、香料味，似乎还有一点咖啡，以及从始至终都有的香水味一直在萦绕。整个过程各种味道复合在一起，层次不是特别明显，整体非常融合，酒精刺激度很低，但也有灼热的一面。
C：这个酒，对我来说又是一次新的味觉体验，它包含了格兰杰的香水味贯穿全程，又有花格那种特别平衡的雪莉底子，同时，酒体也十分活泼，一入口便马上在口腔中铺开，一气呵成，非常不错。
